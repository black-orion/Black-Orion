Black - Orion
=============

A emulator for fly for fun with some tools, based form spheron rev 6 .

Language Used
-------------

Black-Orion - C++
Loader Black-orion - C#
Ultime Black Orion - C#
Tools MJ - C#

Pre-Request
-----------

* Visual studio 2008:
	* For Build source code

* MySql 4 or better:
	* For run sql part
	
* Notepad++ / Hexedit:
 	* For edit resources


Platforme
---------

Run on Windows(XP/7) and Linux
